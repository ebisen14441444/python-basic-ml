# 15. 機械学習講習会への接続

## 機械学習コードを読む前に

機械学習のコードは、だいたい次の流れで書かれます。

1. データを読み込む
2. データを確認する
3. 入力 `X` と正解 `y` に分ける
4. 学習用とテスト用に分ける
5. モデルを作る
6. 学習する
7. 予測する
8. 評価する

この基礎講習会では、特に 1 から 3 までをスムーズに読める状態を目指します。

## データを見る視点

機械学習では、いきなりモデルを使う前にデータを確認します。

Polarsでは、次のような操作で表の様子を確認できます。

```python
print(df.head())
print(df.shape)
print(df.columns)
print(df.schema)
```

確認したいことは主に次の4つです。

- 行と列はいくつあるか
- どんな列があるか
- 数値列と文字列列はどれか
- 正解として予測したい列はどれか

## 入力と正解

例えば、数学と英語の点数から合格したかどうかを予測したいとします。

```python
X = df.select(["math", "english"])
y = df.select("passed")
```

- `X`: モデルに渡す材料
- `y`: モデルに当ててほしい答え

`X` は複数の列を持つ表、`y` は正解ラベルの列です。

## NumPyとPolarsの役割

NumPyとPolarsは、どちらもデータを扱うために使います。

- NumPy: 数値のまとまりを計算する
- Polars: 表形式データを読み込み、加工する

機械学習では、Polarsで表を整えて、必要に応じてNumPy配列に変換してモデルに渡すことがあります。
